end-to-end encrypted
cloud file storage
and collaboration protocol

hengdang is an open protocol for storing your data in an end to end encrypted manner, where even your metadata is hidden, in a third part cloud storage, with file sharing and collaborative capabilities

features

end to end encrypted

all the files and metadata are e2ee on the user's computer

self hostable

host your own storage servers, or choose from a marketplace

file sharing

share files with other users, on any servers, by using their public keys

collaboration apps

collaboratively work on files such as docs or sheets, while preserving privacy

interoperable

apps and different servers all work with each other, on all platforms

open protocol

the protocol is open for anyone to build upon. the reference implementations are foss as well

value proposition

what is hengdang

Section 1

  • experience unparalleled privacy with client-side encryption, ensuring that your data remains private and secure. Neither the service providers nor third parties can access your files without your explicit permission.
  • use any client app on the web, mobile or desktop, and choose any server as your storage provider. everything is interoperable due to hengdang
